## 📂 Danh sách Project

## 1️⃣ HaimiyaMio  

Thư mục **HaimiyaMio** gồm 5 bài thực hành thiết kế giao diện với Android Studio:

### 📌 Nội dung

- Bài 1: Giao diện Thông tin cá nhân (LinearLayout)  
- Bài 2: Form Đăng nhập (ConstraintLayout + Guideline)  
- Bài 3: Danh sách môn học (ScrollView + CardView)  
- Bài 4: Profile sử dụng Fragment (không tạo Activity mới)  
- Bài 5: Mini Project – Ứng dụng Quản lý lớp học  

### 🛠 Công nghệ sử dụng

- Java  
- XML Layout  
- Android Studio  
- Fragment  
- Material Components  

> Lưu ý: Project tập trung vào thiết kế giao diện (UI), chưa xử lý dữ liệu hoặc database.

---

## 2️⃣ Laptrinhsukien  

Thư mục **Laptrinhsukien** gồm 4 bài thực hành về kỹ thuật lập trình sự kiện trong Android:

### 📌 Nội dung

- Bài 1: Máy tính 2 số (Basic Calculator)  
- Bài 2: Tính điểm trung bình  
- Bài 3: Chuyển đổi nhiệt độ (°C ⇄ °F)  
- Bài 4: Giao diện đăng nhập và điều hướng giữa các chức năng  

### 🛠 Công nghệ sử dụng

- Java  
- XML Layout  
- Android Studio  
- Intent & Activity  
- Event Handling  

> Lưu ý: Project tập trung vào xử lý sự kiện, kiểm tra dữ liệu đầu vào và điều hướng giữa các màn hình.
