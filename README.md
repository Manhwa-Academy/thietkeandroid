## 1️⃣ HaimiyaMio  

📅 Ngày thực hiện: 02/03/2026  

Thư mục **HaimiyaMio** gồm 5 bài thực hành thiết kế giao diện với Android Studio:

### 📌 Nội dung

- Bài 1: Giao diện Thông tin cá nhân (LinearLayout)  
- Bài 2: Form Đăng nhập (ConstraintLayout + Guideline)  
- Bài 3: Danh sách môn học (ScrollView + CardView)  
- Bài 4: Profile sử dụng Fragment (không tạo Activity mới)  
- Bài 5: Mini Project – Ứng dụng Quản lý lớp học  

### 🛠 Công nghệ sử dụng

- Java  
- XML Layout  
- Android Studio  
- Fragment  
- Material Components  

> Project tập trung vào thiết kế giao diện (UI), chưa xử lý dữ liệu hoặc database.

## 2️⃣ Laptrinhsukien  

📅 Ngày thực hiện: 03/03/2026  

Thư mục **Laptrinhsukien** gồm 4 bài thực hành về kỹ thuật lập trình sự kiện trong Android:

### 📌 Nội dung

- Bài 1: Máy tính 2 số (Basic Calculator)  
- Bài 2: Tính điểm trung bình  
- Bài 3: Chuyển đổi nhiệt độ (°C ⇄ °F)  
- Bài 4: Giao diện đăng nhập và điều hướng giữa các chức năng  

### 🛠 Công nghệ sử dụng

- Java  
- XML Layout  
- Android Studio  
- Intent & Activity  
- Event Handling  

> Project tập trung vào xử lý sự kiện, kiểm tra dữ liệu đầu vào và điều hướng giữa các màn hình.

## 3️⃣ 10bai  

📅 Ngày thực hiện: 03/03/2026  

Thư mục **10bai** gồm 10 bài thực hành nâng cao về thiết kế giao diện và xử lý sự kiện trong Android.

### 📌 Nội dung

- Bài 1: Ứng dụng Đăng ký tài khoản nâng cao  
- Bài 2: Quản lý sinh viên (2 màn hình)  
- Bài 3: Ứng dụng Đặt vé xem phim  
- Bài 4: Ứng dụng Tính lương nhân viên  
- Bài 5: Ứng dụng Trắc nghiệm 5 câu  
- Bài 6: Quản lý sản phẩm mini  
- Bài 7: Ứng dụng Đặt phòng khách sạn  
- Bài 8: Quản lý chi tiêu cá nhân  
- Bài 9: Ứng dụng Đăng ký môn học  
- Bài 10: Ứng dụng Bán hàng mini  

### 🛠 Công nghệ sử dụng

- Java  
- XML Layout  
- Android Studio  
- Intent & Activity  
- ListView  
- ArrayList  
- AlertDialog  
- Event Handling  

> Project tập trung vào xử lý sự kiện nâng cao, kiểm tra dữ liệu, tính toán và điều hướng nhiều màn hình.
